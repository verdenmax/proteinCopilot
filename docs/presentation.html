🏗️ 架构设计

用户 ◄─► MCP Client (Copilot CLI / Claude Desktop) │ ├── .github/agents/ ← 领域 Agent（LLM 读取） ├── .github/prompts/ ← 12 个 Skill Prompt │ └── MCP Server (Rust, 25 tools, stdio transport) │ ├── spectrum-io 谱图解析（mgf/mzML, DDA+DIA） ├── param-recommend 参数推荐（规则引擎） ├── search-engine 搜索（SimpleSearch + Sage + pFind） ├── protein-inference 蛋白推断（parsimony + razor） ├── fdr FDR 控制（PSM/肽段/蛋白 三级） ├── xic XIC 色谱图 + SILAC ├── dia-extraction DIA 前体提取 ├── result-import 外部结果导入 ├── fasta-db 数据库管理 ├── report 报告生成 └── core 共享模型 + diagnostics

核心原则：Rust 做所有确定性计算 · LLM 做意图理解和结果解释 · 绝对分层

🔍 搜索诊断（新功能）

🏷️

错误分类

4 类错误分类：InputData · Parameters · Database · Engine
自动标记失败阶段

📏

阶段指标

每个搜索阶段记录耗时和处理量：
fasta_parsing · digestion · matching · fdr

⚠️

7 条异常检测规则

低 ID 率 · 高 FDR · 零 decoy · 窄容差 · 宽容差 · 数据库不匹配 · 搜索过慢

💡

修复建议

分级优先级建议 + 具体参数修改
param_changes JSON 可直接应用

🤖

AI 解释

LLM 读取结构化诊断报告
生成人类可读的分析和建议

🛡️

边界安全

fail_stage 幂等 · 双重 finalize 保护
零 PSM 错误级别处理
